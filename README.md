*ROS2 咖啡店模拟项目 (Customer & Peets)*

📖 项目概述

本项目为ROS2学习实践项目，模拟咖啡店顾客与咖啡师的交互场景。包含两个核心节点：

    顾客节点 (Customer)：生成饮品订单请求

    咖啡师节点 (Peets)：处理订单并返回制作状态

🛠️ 环境配置

    操作系统: Ubuntu 20.04 LTS

    ROS2版本: Foxy Fitzroy

    编译系统: colcon

    依赖项:
    bash
    Copy

    sudo apt install ros-foxy-rclcpp ros-foxy-std-msgs
